# SecureFile

SecureFile is a modern, responsive full-stack web application designed for secure file management.

## Features
- **Modern UI**: A beautifully designed user interface with a "Glassmorphism & Vibrant Gradients" theme, including smooth animations and transitions.
- **Frontend**: Built with React, TypeScript, Vite, and Tailwind CSS.
- **Backend**: Powered by Node.js, Express, and specialized tools for file storage and processing.

## Project Structure
- `frontend/`: Contains the React application code, UI components, pages, and Tailwind CSS configuration.
- `backend/`: Contains the Node.js server, API routes, database integrations, and middleware.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/AbhishekDubey8528/SecureFile.git
   ```
2. Install dependencies for the backend:
   ```bash
   cd backend
   npm install
   ```
3. Install dependencies for the frontend:
   ```bash
   cd ../frontend
   npm install
   ```

### Running the Application Locally
To start the development servers, run the appropriate dev commands for frontend and backend:

**Backend:**
```bash
cd backend
npm run dev
```

**Frontend:**
```bash
cd frontend
npm run dev
```
The frontend proxy configuration automatically handles requests to the backend API.
